# Lab Problems

## List of Problems in Section02 (Image Transformation)

1.	Write a program that demonstrates the histogram of an 8-bit grayscale image.
2.	Write a program that computes a histogram with B = 32 bins for an 8-bit grayscale image with K = 256 intensity levels.
3.	Write a program that reads an image, displays an image, and a constant value to each pixel and display the resultant image.
4.	Write a program that reads two individual images and subtracts one from another. 
5.	Write a program that performs image multiplication and division.
6.	Write a program that inverts the image.
7.	Write a program that reads two images, converts into binary images, and performs XOR operation of them.
8.	Write a program that reads an image, and performs the thresholding operation of the image.
9.	Write a program that reads an image and performs logarithmic transformation.
10.	Write a program that reads an image, and performs exponential transformation.
11.	Write a program that differentiates a bad image from a good image based on histogram. 
12.	Write a program that performs contrast stretch using Method 1. (Compute the histogram of the input image and select c and d as the 5th and 95th percentile points of the cumulative distribution (i.e. 5% of the image pixels will be less than c and 5% greater than d).
13.	Write a program that performs histogram equalizations.
14.	Write a program that performs histogram matching.

## List of Problems in Section03 (Image Enhancement)

1.	Write a program or script to apply a 3×3 standard mean filter to an image corrupted by additive white Gaussian noise and salt and pepper noise.
2.	Write a program or script to apply a 3×3 standard median filter to an image corrupted by additive white Gaussian noise and salt and pepper noise.
3.	Write a program or script to apply order filtering (max, order = 25, 5×5) to an image corrupted by additive white Gaussian noise and salt and pepper noise.
4.	Write a program that uses the Gaussian filter with different standard deviations (σ = 1, σ = 3) to an image corrupted by additive white Gaussian noise and salt and pepper noise and observe the changes in image sharpness.

## List of Problems in Section04 (Image Compression)

1.	Write a program to measure and calculate the spatial, physical, and informational properties of a given digital image.
2.	To implement and analyze the Huffman Coding Algorithm as a lossless image compression technique, evaluate its compression performance, and verify perfect reconstruction through decoding.

## List of Problems in Section05 (Image Segmentation)

1.	Implement, compare, and analyze the performance of first-order derivative and second-order derivative operators for edge detection in digital images.
2.	To implement and evaluate an isolated point detection algorithm using the Laplacian mask.
3.	Load a standard grayscale test image or any high-contrast image and apply the following edge detection filters. (a) Roberts Cross Operator: Apply the 2×2 convolution kernels to compute the diagonal gradients. (b) Prewitt Operator: Apply the 3 × 3 separable kernels to detect horizontal and vertical edges. (c) Sobel Operator: Apply the 3 × 3 weighted kernels to calculate the exact gradient magnitude and direction.
